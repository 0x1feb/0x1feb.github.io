Ubuntu 22.04 LTS に NGINX をインストールする

概要#

  • Ubuntu 22.04 LTS に NGINX をインストールする。

前提#

  • Ubuntu 22.04 LTS
  • NGINX 1.26.0

手順#

Ubuntu Repository 経由#

この手順では、Ubuntu Repository の最新版である 1.18.0 がインストールされる。

NGINX をインストールする。

sudo apt install nginx

NGINX のバージョンを確認する。

$ sudo nginx -v
nginx version: nginx/1.18.0 (Ubuntu)

Official NGINX Repository 経由#

この手順では、Official NGINX Repository の最新版である 1.26.0 がインストールされる。

前提となるアプリをインストールする。

sudo apt install curl gnupg2 ca-certificates lsb-release ubuntu-keyring

公式の NGINX 署名鍵をインポートする。

curl https://nginx.org/keys/nginx_signing.key | gpg --dearmor \
| sudo tee /usr/share/keyrings/nginx-archive-keyring.gpg >/dev/null

ダウンロードした鍵が適切か確認する。

gpg --dry-run --quiet --no-keyring --import --import-options import-show /usr/share/keyrings/nginx-archive-keyring.gpg

573BFD6B3D8FBC641079A6ABABF5BD827BD9BF62 であれば問題ない。

pub   rsa2048 2011-08-19 [SC] [expires: 2024-06-14]
      573BFD6B3D8FBC641079A6ABABF5BD827BD9BF62
uid                      nginx signing key <signing-key@nginx.com>

apt レポジトリに安定板の NGINX を追加する。

echo "deb [signed-by=/usr/share/keyrings/nginx-archive-keyring.gpg] \
http://nginx.org/packages/ubuntu `lsb_release -cs` nginx" \
    | sudo tee /etc/apt/sources.list.d/nginx.list

追加したパッケージを優先するよう設定する。

echo -e "Package: *\nPin: origin nginx.org\nPin: release o=nginx\nPin-Priority: 900\n" \
    | sudo tee /etc/apt/preferences.d/99nginx

NGINX をインストールする。

sudo apt update
sudo apt install nginx

NGINX のバージョンを確認する。

$ sudo nginx -v
nginx version: nginx/1.26.0

動作確認#

NGINX を起動する。

sudo systemctl start nginx

ローカルホストに接続する。

$ curl -I 127.0.0.1
HTTP/1.1 200 OK
Server: nginx/1.26.0
Date: Wed, 24 Apr 2024 14:34:24 GMT
Content-Type: text/html
Content-Length: 615
Last-Modified: Tue, 23 Apr 2024 14:04:32 GMT
Connection: keep-alive
ETag: "6627bff0-267"
Accept-Ranges: bytes

ブラウザで http://{hostname} を開くと、「Welcome to nginx!」ページが表示される。

OS の起動時に NGINX サーバが動くようにする。

sudo systemctl enable nginx
