Jakarta Faces 4.0 で JavaScript を defer/async で読み込む

概要#

  • Jakarta Faces 4.0 の XHTML にて、<h:outputScript> に defer/async 属性を付与できなかった。
  • <script src="..." defer="defer"></script> タグを使って読み込むことで対応した。

前提#

  • Jakarta Faces 4.0

実装#

  • src/main/webapp/resources/js/main.js を defer 属性で読み込む。

修正前#

<h:outputScript library="js" src="main.js" />

なお、以下のような記載にしても、defer や async は反映されなかった。

<h:outputScript library="js" src="main.js" defer="defer" />

修正後#

<script src="#{resource['js/main.js']}" defer="defer"></script>

XHTML では、defer="defer" のように、属性名を明示的に指定する必要がある。
