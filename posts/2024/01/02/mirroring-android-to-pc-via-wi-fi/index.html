Wi-Fi 経由で Android 端末を PC へミラーリングする

概要#

  • Wi-Fi 経由で Android 端末を PC へミラーリングする。
    • Android 端末の画面をキャプチャできる。
    • PC から Android 端末を操作できる。

前提#

  • Android 端末: Pixel 6a (Android 11)
  • PC: Windows 11
  • Software: scrcpy 2.3.1

手順#

PC に scrcpy をインストールする#

  1. PC にて、scrcpy (Release scrcpy v2.3.1 · Genymobile/scrcpy) をインストールする。

    • Windows は winget を使ったインストールもできる。

Android の Wi-Fi デバッグ機能を有効化する#

  1. Settings > About phone > Build number より、ビルド番号を7回タップする。
  2. Settings > System > Developer options をタップする。
  3. Wireless debugging を有効化する。

ペアリングして接続する#

  1. Android 端末にて、Settings > System > Developer options をタップする。

  2. Pair device with pairing code をタップする。

    • 表示されている情報を次の手順で使用する。

  3. PC にて、Android 端末に表示されている情報でペアリングする。

    > adb pair 192.168.11.1:12345 999999
Successfully paired to 192.168.11.1:12345 [guid=adb-HOGE-FUGA]

  4. ペアリングした端末に接続する。

    • ペアリング用に表示されたものではなく、Android 端末の Wireless debugging 画面に表示されている IP Address & Port なので注意する。
    > adb connect 192.168.11.1:67890
connected to 192.168.11.1:67890
