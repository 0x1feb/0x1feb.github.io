解説#

@Test#

@Test アノテーションは、付与したメソッドがテストメソッドであることを表す。

以下の ExampleTest は、@Test のテストを 2 つ持つ。

class ExampleTest {
    @Test
    void testMethod1() {
        assertEquals(2, 1 + 1);
    }

    @Test
    void testMethod2() {
        assertEquals(2, 1 + 1);
    }
}

このテストクラスでは、testMethod1 と testMethod2 の 2 つが実行される。

ExampleTest: 2 total, 2 passed

ExampleTest
  testMethod1 [passed]
  testMethod2 [passed]
