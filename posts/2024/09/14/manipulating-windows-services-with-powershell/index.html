Windows のサービスを PowerShell のコマンドで操作する

概要#

  • Windows のサービスを PowerShell のコマンドで操作する。
  • サービスのスタートアップの種類を変更する。

前提#

  • PowerShell 7.4

説明#

サービスの取得#

以下のコマンドを実行すると、全サービスを取得する。

Get-Service

個別に取得する場合は、-Name か -DisplayName オプションで指定する。

例えば、Windows Update のサービスを -Name で取得する場合、以下のように指定する。

Get-Service -Name "wuauserv"

もしくは、-DisplayName に指定する。

Get-Service -DisplayName "Windows Update"

どちらもワイルドカードを使用できるため、以下のように “windows” を名前に含むサービス一覧を取得することもできる。

Get-Service -DisplayName "*windows*"

サービスの変更#

表示名#

ExampleService の表示名を New Service Display Name に変更する場合、以下のコマンドを実行する。

Set-Service -Name ExampleService -DisplayName "New Service Display Name"

スタートアップの種類#

ExampleService のスタートアップの種類を自動に変更する場合、以下のコマンドを実行する。

Set-Service -Name ExampleService -StartupType Automatic

指定できる StartupType は以下の通りである。

  • Automatic: 自動
  • AutomaticDelayedStart: 自動 (遅延開始)
  • Disabled: 無効
  • Manual: 手動

サービスのステータス変更#

サービスの開始#

ExampleService を開始する場合、以下のコマンドを実行する。

Set-Service -Name ExampleService -Status Running -PassThru

-PassThru パラメータを付与すると、実行結果をコンソールに出力する。

サービスの一時停止#

ExampleService を一時停止する場合、以下のコマンドを実行する。

Get-Service -Name ExampleService | Set-Service -Status Paused

サービスの停止#

ExampleService を停止する場合、以下のコマンドを実行する。

Get-Service -Name ExampleService | Set-Service -Status Stopped
