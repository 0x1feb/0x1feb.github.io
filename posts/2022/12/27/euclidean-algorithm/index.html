最大公約数 (ユークリッドの互除法)

2022-12-27 · 1 min · Me | Suggest Changes
Table of Contents
  • ソースコード
  • 参考

ユークリッドの互除法とは、2つの自然数の最大公約数を求めるアルゴリズムである。

ソースコード#

#include <bits/stdc++.h>
using namespace std;

int gcd(int a, int b) {
  return (b == 0 ? a : gcd(b, a % b));
}

int main() {
  cout << gcd(1071, 1029) << endl;
  cout << gcd(1029, 1071) << endl;
  return 0;
}
