Ubuntu Server 22.04 LTS に Podman をインストールする

概要#

  • Ubuntu Server 22.04 LTS に Podman をインストールする。
  • Podman で作成したコンテナをサービスとして実行できるようにする。

環境#

  • Ubuntu Server 22.04 LTS
  • podman version 3.4.4

手順#

インストール#

以下のコマンドを実行し、Podman をインストールする。

sudo apt install podman

インストールが完了したら、バージョンを確認する。

podman --version

バージョンが表示されたらインストール完了となる。

podman version 3.4.4
