Open WebUI のコンテナをアップデートする

概要#

  • Open WebUI のコンテナをアップデートする。

前提#

  • Ubuntu 24.04 LTS
  • podman version 4.9.3

手順#

既存コンテナの削除#

コンテナが起動している場合、停止する。

podman stop open-webui

現在のコンテナを削除する。

podman rm open-webui

Open WebUI で保存していたデータも削除する場合は、以下を実行する。

podman volume rm open-webui

最新イメージの取得#

最新イメージを取得する。

podman pull ghcr.io/open-webui/open-webui:main

コンテナの起動#

Open WebUI のコンテナを起動する。以下は実行コマンドの例である。

podman run -d -p 3000:8080 \
  --gpus=all \
  -v open-webui:/app/backend/data \
  -e OLLAMA_BASE_URL=http://192.168.1.1:11434 \
  --name open-webui --restart always \
  ghcr.io/open-webui/open-webui:main
