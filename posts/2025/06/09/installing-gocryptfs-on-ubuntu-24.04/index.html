Ubuntu 24.04 に gocryptfs をインストールして暗号化したファイルシステムを構築する

概要#

  • Ubuntu 24.04 に gocryptfs をインストールする。
  • gocryptfs とは、マウント可能な FUSE ファイルシステムとして実装された、ファイルベースの暗号化ソフトウェアである。

環境#

  • Ubuntu 24.04
  • gocryptfs 2.4.0

手順#

以下を参考にインストールや設定をした。

  • Quickstart - gocryptfs

インストール#

Ubuntu にて次のコマンドを実行し、gocryptfs をインストールする。

sudo apt install gocryptfs

バージョンを確認する。

gocryptfs -version

gocryptfs 2.4.0; go-fuse 2.4.2; 2024-11-01 go1.22.2 linux/amd64

設定#

forward mode#

forward mode は、ファイルをディスク上に暗号化して保存し、マウントされたファイルシステムが平文ビューを提供するモードである。

  • Cryptography - gocryptfs

暗号化したファイルが配置される cipher ディレクトリを作成する。

mkdir cipher

cipher ディレクトリを初期化する。

gocryptfs -init cipher

Choose a password for protecting your files.
Password:
Repeat:

パスワードを聞かれるため、任意のパスワードを入力する。

Your master key is:

    xxxxxxxx-xxxxxxxx-xxxxxxxx-xxxxxxxx-
    xxxxxxxx-xxxxxxxx-xxxxxxxx-xxxxxxxx

If the gocryptfs.conf file becomes corrupted or you ever forget your password,
there is only one hope for recovery: The master key. Print it to a piece of
paper and store it in a drawer. This message is only printed once.
The gocryptfs filesystem has been created successfully.
You can now mount it using: gocryptfs cipher MOUNTPOINT

マスターキーが表示されるため、保存する。gocryptfs.conf が破損した場合や、パスワードを忘れた場合、マスターキーがあればリカバリできる。

次のコマンドを実行し、cipher ディレクトリの中身を確認する。

ls cipher/

gocryptfs.conf  gocryptfs.diriv

gocryptfs.conf と gocryptfs.diriv が作成されている。暗号化されたデータ管理と復号に必要なメタデータを保存するファイルである。

復号した平文ファイル群をマウントする plain ディレクトリを作成する。

mkdir plain

次のコマンドを実行し、cipher を plain にマウントする。

gocryptfs cipher plain

Password:

パスワードが求められるため、cipher を初期化したときのパスワードを入力する。

Decrypting master key
Filesystem mounted and ready.

マウントが完了する。

plain に適当なファイルを作成する。

echo hello > plain/test.txt

cipher 配下を確認すると、ファイル名を含めて暗号化された FLYcsos-pYy4AFwLqD6IiQ が作成されている。

ls cipher/
FLYcsos-pYy4AFwLqD6IiQ  gocryptfs.conf  gocryptfs.diriv

中身も暗号化されている。

次のコマンドを実行し、アンマウントする。

fusermount -u plain

これでマウント時以外は暗号化した cipher ディレクトリのみとなるため、ファイルを暗号化した状態で安全に保管できる。

reverse mode#

reverse mode は、ファイルをディスク上に平文で保存し、マウントされたファイルシステムが暗号化ビューを提供するモードである。

  • Reverse Mode Cryptography - gocryptfs

Reverse Mode は、-reverse オプションをつけて、plain と cipher を入れ替えれば良い。

mkdir cipher plain
gocryptfs -reverse -init plain
gocryptfs -reverse plain cipher

cipher をクラウドストレージ等に保管すれば、ファイルを安全にバックアップできる。
