SharedOS で HDD/SSD のデータを一括で完全に消去する

2025-05-09 · 2 min · Me | Suggest Changes

概要#

  • 通常のファイル削除やディスクのフォーマットでは、データを復元できてしまうことがある。
  • ShredOS を使って、復元を困難にする方法で HDD/SSD のデータを消去する。

環境#

  • Windows 11 Pro
  • rufus 4.7
  • shredos.x86_64 v0.38

手順#

shredos.x86_64 の 実行用ディスク作成#

以下の手順は Windows 11 にて実施した。

  1. PartialVolume/shredos.x86_64 より、ShredOS .img x86_64bit for USB Vanilla をダウンロードする。
  2. Rufus より、rufus-4.7.exe (最新版) をダウンロードする。
  3. rufus-4.7.exe を実行する。
  4. 以下を指定し、スタート を選択する。
    • デバイス: USB メモリを指したドライブ
    • ブートの種類: ダウンロードした ShredOS の .img ファイル
  5. 書き込みが完了したら、閉じる を選択する。

ShredOS の実行#

SharedOS が削除対象のディスクを認識できるような構成のコンピュータであれば、どのような環境でも使えるはず。

  1. ShredOS を書き込んだ USB メモリを削除したいディスクが搭載された PC に接続する。
  2. PC を起動する。
  3. BIOS より、USB メモリの SharedOS を起動する。
  4. フォーマットしたいディスクや削除方法を選択する。(詳細は shredos.x86_64/README.md 参照)
  5. 削除が完了したら、再起動またはシャットダウンする。
