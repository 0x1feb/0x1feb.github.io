Pixel 3a に LineageOS 22.2 (Android 15) をインストールする

概要#

  • Pixel 3a は Android 12 が最新版となり、以降のバージョンアップは提供されていない。
  • LineageOS 22.2 をインストールすることで、 Android 15 に対応することができる。

環境#

  • Windows 11 (作業用)
  • Pixel 3a (Android 12)
  • LineageOS 22.2 (Android 15)

手順#

Install LineageOS on sargo ｜ LineageOS Wiki の手順に沿って進める。

事前確認#

Pixel 3a にて、次の点を確認する。

  • 必要なデータを端末外にバックアップしていること。
  • バージョンが Android 12 であること。
  • Factory reset protection を避けるため、端末に登録した Google アカウントが全て削除 (未登録状態) されていること。

ダウンロード#

以下より、Latest の資材をダウンロードする。

  • LineageOS Downloads
    • lineage-22.2-20250510-nightly-sargo-signed.zip
    • boot.img

以下より、Google apps (LineageOS 22 ARM64)をダウンロードする。

  • Google apps ｜ LineageOS Wiki
    • MindTheGapps-15.0.0-arm64-20250214_082511.zip

ダウンロードした資材を任意のパス (今回は D:\tmp 配下) に配置する。

adb と fastboot のインストール#

Windows 11 端末にて、以下の手順を実施する。

  1. platform-tools-latest-windows.zip をダウンロードする。
  2. 任意のパスに展開する。
  3. 展開したパスを環境変数 PATH に追加する。

Google USB Driver のインストール#

Windows 11 端末にて、以下の手順を実施する。

  1. Get the Google USB Driver より、Google USB Driver をインストールする。
    • 直接ドライバをインストールするか、Android SDK Manager 経由でインストールする。
    • Android SDK Manager の方が最新版への更新が簡単に管理できそう。
    • 今回は Android SDK Manager 経由でインストールするため、Android Studio をインストールする。
  2. Download Android Studio & App Tools - Android Developers より、android-studio-2024.3.2.14-windows.exe をダウンロードする。
  3. android-studio-2024.3.2.14-windows.exe を実行し、インストールする。
  4. Android Studio を開く。
  5. Settings > Languages & Frameworks > Android SDK の SDK Tools タブより、Google USB Driver をインストールする。

USB デバッグの有効化#

Pixel 3a にて、以下の手順を実施する。

  1. Settings > About Phone を開く。
  2. Build number を 7 回タップし、開発者モードを有効化する。
  3. Settings > System > Developer options を開く。
  4. Debugging にある USB Debugging を有効化する。

adb のセットアップ#

  1. Windows 11 端末と Pixel 3a を USB ケーブルで接続する。

  2. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    adb devices

  3. Pixel 3a にて、Allow USB debugging? と表示されるため、Always allow from this computer にチェックを入れ、Allow を選択する。

  4. Windows 11 のデバイスマネージャにて、Pixel 3a が表示されていることを確認する。

ブートローダのアンロック#

  1. Pixel 3a にて、Settings > System > Developer options を開く。

  2. OEM unlocking を有効化する。

  3. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    adb -d reboot bootloader

  4. Pixel 3a が再起動し、Fastboot Mode と表示される。

  5. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    fastboot devices
fastboot flashing unlock

    fastboot devices で何も表示されなかったり、エラーが出る場合はデバイスマネージャより、Pixel 3a が正しく認識されているか確認する。 ドライバがインストールされていない場合、 ${env:LOCALAPPDATA}\Android\Sdk\extras\google\usb_driver をインストールする。

  6. Pixel 3a にて、注意書きが表示されるため、内容を確認する。

  7. 音量ボタンで Unlock the bootloader を選択し、電源ボタンで実行する。

Lineage Recovery のインストール#

Windows 11 端末と Pixel 3a を USB で接続し、Pixel 3a が Fastboot Mode で起動した状態で以下の手順を実施する。

  1. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    fastboot flash boot D:\tmp\boot.img

    Sending 'boot_b' (65536 KB)                        OKAY [  1.689s]
Writing 'boot_b'                                   OKAY [  0.615s]
Finished. Total time: 2.667s

  2. Pixel 3a にて、音量ボタンで Recovery Mode を選択し、電源ボタンで実行する。

  3. 再起動し、RECOVERY と書かれた画面が表示される。

LineageOS のインストール#

Windows 11 端末と Pixel 3a を USB で接続し、Pixel 3a が Recovery Mode で起動した状態で以下の手順を実施する。

  1. Pixel 3a にて、Factory reset を選択する。

  2. Format data/factory reset を選択する。

  3. ダイアログが表示されるため、Format data を選択する。

  4. Data wipe complete. と表示されたら、フォーマット完了となる。

  5. メインメニューに戻り、Apply Update > Apply from ADB を選択する。

  6. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    adb -d sideload D:\tmp\lineage-22.2-20250510-nightly-sargo-signed.zip

  7. 処理が 47% で止まることがあるが、特に問題なく進んでいる。

    serving: 'D:\tmp\lineage-22.2-20250510-nightly-sargo-signed.zip'  (~47%)

  8. Pixel 3a にて、Do you want to reboot to recovery now? と表示されるため、Yes を選択する。

  9. Windows 11 端末側にて、次のいずれかの結果が出ることがあるが、問題ない。

    adb: failed to read command: Success

    adb: failed to read command: No error

    adb: failed to read command: Undefined error: 0

アドオンのインストール#

  1. Pixel 3a にて、Apply Update > Apply from ADB を選択する。

  2. Windows 11 端末にて PowerShell を開き、次のコマンドを実行する。

    adb -d sideload D:\tmp\MindTheGapps-15.0.0-arm64-20250214_082511.zip

  3. Signature verification failed と表示されるため、Yes を選択する。

  4. メインメニューに戻る。

システムの起動#

  1. Pixel 3a にて、Reboot system now を選択する。
  2. 数分待つと、初期設定が始まるため、セットアップする。 通常は 15 分以内に起動するため、もし 15 分以上経っても起動しない場合は手順にミスがあった可能性がある。
  3. 設定が完了したら、Settings > About Phone より、Android version と LineageOS version がインストールしたバージョンになっていることを確認する。
