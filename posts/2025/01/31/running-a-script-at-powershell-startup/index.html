PowerShell の起動時に自動でスクリプトを実行する

概要#

  • PowerShell のプロファイルを作成する。
  • プロファイルに任意のスクリプトを記載する。

前提#

  • PowerShell 7.5.0

手順#

プロファイルパスの確認#

以下コマンドを実行し、プロファイルのパスを確認する。

$PROFILE | Select-Object *

AllUsersAllHosts       : C:\Program Files\PowerShell\7\profile.ps1
AllUsersCurrentHost    : C:\Program Files\PowerShell\7\Microsoft.PowerShell_profile.ps1
CurrentUserAllHosts    : C:\Users\{user}\Documents\PowerShell\profile.ps1
CurrentUserCurrentHost : C:\Users\{user}\Documents\PowerShell\Microsoft.PowerShell_profile.ps1
Length                 : 74

$PROFILE 変数は、以下の値を持っている。

  • $PROFILE: 現在のユーザ/現在のホスト
  • $PROFILE.CurrentUserCurrentHost: 現在のユーザ/現在のホスト
  • $PROFILE.CurrentUserAllHosts: 現在のユーザ/すべてのホスト
  • $PROFILE.AllUsersCurrentHost: すべてのユーザ/現在のホスト
  • $PROFILE.AllUsersAllHosts: すべてのユーザ/すべてのホスト

プロファイルの作成#

以下コマンドを実行し、現在のユーザ/現在のホストの PowerShell のプロファイルパスを確認する。

echo $PROFILE

C:\Users\{user}\Documents\PowerShell\Microsoft.PowerShell_profile.ps1

プロファイルパスに対応するファイルが存在しない場合、作成する。

if (!(Test-Path $PROFILE)) { New-Item -ItemType File -Path $PROFILE -Force }

    Directory: C:\Users\{user}\Documents\PowerShell

Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
-a---      2025-01-31(金)    20:23              0 Microsoft.PowerShell_profile.ps1

プロファイルの編集#

$PROFILE のパスにあるファイルを任意のエディタで開く。

実行したいスクリプトを記述する。

Write-Host "hello, world!"

UTF-8 with BOM で保存する。

確認#

PowerShell を起動すると、記載したスクリプトが実行される。

hello, world!

Import-Module 等で自作のモジュールを読み込むようにすれば、PowerShell で常に自作の関数を使うことができる。
