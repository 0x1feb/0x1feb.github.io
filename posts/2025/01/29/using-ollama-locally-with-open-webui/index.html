Open WebUI で Ollama のローカル LLM をブラウザから使う

概要#

  • Open WebUI を WSL2 の Podman で動かす。
  • ローカルで動かしている Ollama を Open WebUI から呼び出す。

環境#

  • ollama version is 0.5.7
  • Podman (Docker)
  • NVIDIA GeForce RTX 3060 12GB

手順#

Ollama のインストール#

以下より、Ollama をダウンロードし、インストールする。

  • Ollama

今回は Open WebUI と Ollama が別ネットワークで動かすため、Ollama を外から接続できるようにする。

  • ollama/docs/faq.md at main · ollama/ollama · GitHub

上記 QA を参考に、以下の環境変数を追加する。

  • OLLAMA_HOST: 0.0.0.0

Open WebUI のインストール#

以下 URL の説明を参考に、docker コマンドを podman コマンドにして実行する。

podman run -d -p 3000:8080 -v open-webui:/app/backend/data -e OLLAMA_BASE_URL=http://192.168.1.1:11434 --name open-webui --restart always ghcr.io/open-webui/open-webui:main

このとき、-e LLAMA_BASE_URL=http://192.168.1.1:11434 にて、Ollama を動かしているサーバを指定する。

Open WebUI の操作#

正常に起動していれば、ブラウザから以下 URL で接続できる。

  • http://localhost:3000

最初にアカウント作成画面が表示されるため、ログイン用の適当なアカウントを作成する。ローカルかつ個人でしか利用しないのであれば、適当な値で問題ない。以後のログインで使用するため、作成したアカウント情報は記録しておく。

ログイン後、ChatGPT のような UI が表示される。

左上の選択肢にて、Ollama に設定済みのモデルが選択できる。

後は普通にチャットで質問すれば、指定したモデルでの回答が表示される。
