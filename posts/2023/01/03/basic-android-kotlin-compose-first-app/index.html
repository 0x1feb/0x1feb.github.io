概要#

Android アプリを作ってみたくなったので、チュートリアルを試してみた結果をメモとして残す。

準備#

  • Kotlin の基礎知識があること。
  • Android Studio がインストールされていること。
  • 実機でアプリを動かす場合は確認用の Android 端末があること。

サンプルアプリの作成#

テンプレートからプロジェクトを作成する#

  1. Android Studio を起動し、[New Project] をクリックする。
  2. [Phone and Tablet] を選択し、[Empty Compose Activity] を選択する。
  3. [Next] をクリックする。
  4. 以下を入力する。
    • Name: GreetingCard
    • Package name: com.example.greetingcard
    • Save location: 任意のパス
    • Language: Kotlin (固定)
    • Minimum SDK: API 21: Android 5.0 (Lollipop)
  5. [Finish] をクリックする。

読み込みが終わるまで時間がかかるのでしばらく待つと、テンプレートを使用したプロジェクトが作成される。

テキスト/デザインを更新する#

  1. MainActivity.kt の Greeting() を次のように変更する。

    @Composable
fun Greeting(name: String) {
  Text(text = "Hi, my name is $name!")
}

  2. Greeting() を呼び出すときの引数を任意の名前に変更する。

    Greeting("rano")

  3. Text... の行をハイライト表示し、Alt + Enter を押して、[Surround with widget] を選択する。

    @Composable
fun Greeting(name: String) {
    Box {
        Text(text = "Hi, my name is $name!")
    }
}

  4. Box を Surface() に変更する。

    @Composable
fun Greeting(name: String) {
    Surface() {
        Text(text = "Hi, my name is $name!")
    }
}

  5. color パラメータを追加する。（ここでは青を指定）

    ...
import androidx.compose.ui.graphics.Color
...
@Composable
fun Greeting(name: String) {
    Surface(color = Color.Blue) {
        Text(text = "Hi, my name is $name!")
    }
}

  6. パディングを追加する。

    ...
import androidx.compose.foundation.layout.padding
...
import androidx.compose.ui.unit.dp
...
@Composable
fun Greeting(name: String) {
    Surface(color = Color.Blue) {
        Text(text = "Hi, my name is $name!", modifier = Modifier.padding(24.dp))
    }
}

以上で簡単なカスタマイズが完了した。

実機での動作確認#

開発用デバイスをセットアップする#

デバイスとの接続は、USB 接続と Wi-Fi 接続の2種類あるが、今回は Wi-Fi 経由で接続する。

  1. デバイスの設定アプリより、[開発者向けオプション] を選択し、[ワイヤレス デバッグ] を有効にする。
    • デバイスの開発者向けオプションを設定する  ｜  Android デベロッパー  ｜  Android Developers

デバイスに接続する#

  1. 実行構成のプルダウンメニューから [Pair Devices Using Wi-Fi] を選択する。
  2. デバイスの設定アプリより、[開発者向けオプション] を選択し、[ワイヤレス デバッグ] を選択する。
  3. [QR コードによるデバイスのペア設定] を選択し、QR コードを読み取る。
  4. ペアリングしたら対象のデバイスを選択し、[Run ‘app’] のアイコンをクリックする。
  5. 実機端末にサンプルアプリが表示される。

基本はエミュレータでの確認で問題ないと思うが、実際の操作感を確認したい場合は実機を使うのが良い。
