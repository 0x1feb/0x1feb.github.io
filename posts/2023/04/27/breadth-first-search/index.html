幅優先探索 (Breadth-first search / BFS)

概要#

幅優先探索とは、グラフの探索等に用いられる探索アルゴリズムである。ある頂点をスタートとし、その点に隣接する全ての点を探索する。探索したそれぞれの点についても同様に隣接する点を探索することを繰り返す。

以下のような各頂点に番号が振られているグラフを例に考える。

graph TD
    0((0))---1((1))
    0((0))---2((2))
    0((0))---3((3))
    1((1))---4((4))
    1((1))---5((5))
    3((3))---6((6))
    4((4))---7((7))
    4((4))---8((8))
    6((6))---9((9))

入力は以下の形式で標準入力から与えられるものとする。

$N ~ M ~ S ~ T \\ A_{1} ~ B_{1} \\ \vdots \\ A_{m} ~ B_{m}$

  • $N$: 頂点の数
  • $M$: 辺の数
  • $S$: 開始頂点
  • $T$: 終了頂点
  • $A_{i} ~ B_{i}$: $i$ 本目の辺は頂点 $A_{i}$ と $B_{i}$ を結んでいる

頂点 $S$ から頂点 $T$ までの経路を出力する。例えば、頂点8から頂点9までの経路は、以下の太線 (8 4 1 0 3 6 9) となる。

graph TD
    0((0))===1((1))
    0((0))---2((2))
    0((0))===3((3))
    1((1))===4((4))
    1((1))---5((5))
    3((3))===6((6))
    4((4))---7((7))
    4((4))===8((8))
    6((6))===9((9))

ソースコード#

#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (n); ++i)
#define rng(a) a.begin(), a.end()
using vi = vector<int>;
using vvi = vector<vi>;
using vb = vector<bool>;

int main() {
  int n, m, s, t;
  cin >> n >> m >> s >> t;
  vvi g(n);
  rep(i, m) {
    int a, b;
    cin >> a >> b;
    g[a].push_back(b);
    g[b].push_back(a);
  }

  vi prev(n, -1);
  vb visited(n, false);
  queue<int> que;
  que.push(s);
  while (!que.empty()) {
    int v = que.front();
    que.pop();
    visited[v] = true;
    for (int v_next : g[v]) {
      if (visited[v_next]) continue;
      prev[v_next] = v;
      que.push(v_next);
    }
  }

  vi path;
  int u = t;
  while (u != -1) {
    path.push_back(u);
    u = prev[u];
  }
  reverse(rng(path));

  int path_size = path.size();
  rep(i, path_size) cout << path[i] << (i < path_size - 1 ? " " : "\n");
  return 0;
}

// 入力
10 9 8 9
0 1
0 2
0 3
1 4
1 5
3 6
4 7
4 8
6 9

// 出力
8 4 1 0 3 6 9

解説#

まず、入力値を受け取る。

  int n, m, s, t;
  cin >> n >> m >> s >> t;
  vvi g(n);
  rep(i, m) {
    int a, b;
    cin >> a >> b;
    g[a].push_back(b);
    g[b].push_back(a);
  }

2次元配列 g を作成し、 辺の情報を格納する。g[x] は、点 x と繋がる点のリストとなる。

次に、探索する。

  vi prev(n, -1);
  vb visited(n, false);
  queue<int> que;
  que.push(s);
  while (!que.empty()) {
    int v = que.front();
    que.pop();
    visited[v] = true;
    for (int v_next : g[v]) {
      if (visited[v_next]) continue;
      prev[v_next] = v;
      que.push(v_next);
    }
  }

prev[x] には、点 x に到達したときに経由した1つ前の点を記録する。点 x から点 y を探索したとき、 prev[y] に x を格納する。prev は -1 で初期化し、最初に探索する点の前の点は -1 とする。

visited[x] には、点 x が探索済みかどうかを記録する。点 x を探索したとき、 visited[x] は true となる。

探索は、以下の流れで実行する。

  1. que に探索の開始点 s を追加する。
  2. que が空になるまで以下を繰り返す。
    1. que から先頭の要素を取り出す。
    2. 取り出した点 v を探索済みとして visited に記録する。
    3. v と辺で繋がる点 v_next を取り出す。
    4. v_next が探索済みであれば、以降の処理はスキップする。
    5. prev にどの点からどの点へと探索したか記録する。
    6. que に v_next を追加する。
  3. que が空になったら処理を終了する。

最後に、探索したパスを出力する。

  vi path;
  int u = t;
  while (u != -1) {
    path.push_back(u);
    u = prev[u];
  }
  reverse(rng(path));

  int path_size = path.size();
  rep(i, path_size) cout << path[i] << (i < path_size - 1 ? " " : "\n");

目的となる点 t の1つ前の点は prev[t] で取得できる。この点を u とすると、 t の2つ前の点は prev[u] で取得できる。これを繰り返して s にたどり着くまでの点の情報を path に格納する。

path には、 t から s までの経路上の点が格納されるため、逆順にしてから出力する。

辺の数だけ探索するため、計算量は $O(M)$ となる。
