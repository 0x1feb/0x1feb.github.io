バケットソート (Bucket Sort)

2023-02-16 · 1 min · Me | Suggest Changes
Table of Contents
  • ソースコード
  • 解説
  • 参考

バケットソートとは、整列したいデータがとりうる値の数だけバケツを用意し、全ての要素をバケツに入れた後、順番に取り出すソートアルゴリズムである。

ソースコード#

#include <bits/stdc++.h>
#define rep(i, n) for (int i = 0; i < (n); ++i)
using namespace std;
using vi = vector<int>;
const int X = 1000000;

void print_list(vi a, int n) {
  rep(i, n) cout << a[i] << (i < n - 1 ? " " : "\n");
}

void bucket_sort(vi &a, int n) {
  // 要素の値の個数を記録する
  vi count(X);
  rep(i, n) count[a[i]]++;
  // 開始位置を記録する
  vi offset(X);
  rep(num, X) offset[num] = (num == 0 ? count[num] : offset[num - 1] + count[num]);

  // ソート
  vi b(n);
  rep(i, n) b[--offset[a[i]]] = a[i];
  a = b;
}

int main() {
  int n;
  cin >> n;
  vi a(n);
  rep(i, n) cin >> a[i];

  bucket_sort(a, n);
  print_list(a, n);
  return 0;
}

実行した結果は以下のとおり。

// 入力
8
2 1 3 7 1 6 2 5

// 出力
1 1 2 2 3 5 6 7

解説#

  1. 配列 a の各要素の個数をカウントする。
    • $i$ 番目の要素の数は count[a[i]] 個となる。
  2. 開始位置を記録する。
    • offset[num] が num 以下の数の要素数となる。
  3. ソート結果を配列 b に格納する。
    • $i$ 番目の要素の開始位置である --offset[a[i]] に a[i] を格納する。

計算量は $O(n^{2})$ となる。要素が $k$ 種類のとき、平均計算量は $O(n + k)$ となる。
